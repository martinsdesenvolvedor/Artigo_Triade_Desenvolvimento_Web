Html, Css e JavaScript: Um pouco da História de cada um

25/09/2022

Html, Css e Javascript são as três principais linguagens utilizadas no Front-End. Elas são utilizadas do lado do cliente, no próprio navegador.

Há um tempo atrás, havia muita dificuldade de compartilhar documentos entre os próprios funcionários de uma empresa e isto demandava tempo. Em 1991, Tim Berners-Lee, projetou o Html, uma linguagem de marcação de texto. No início, o Html era somente para facilitar o compartilhamento de documentos. Com o advento da rede de alcance mundial, world wide web – www, em 1992, o Html passou a ser usada para tudo e por todos.

Antes da criação do Css, os desenvolvedores estilizavam suas páginas no mesmo arquivo da parte estrutural, tornando-os cada vez mais extensos e difíceis de entender. Então, em 1995 o Css é criado para que esta estilização de página web fosse feito em um arquivo separado do Html.

Já o Javascript, foi criado a partir de uma dura concorrência entre as gigantes da época, Microsoft e Netscape. A Netscape criou uma linguagem de servidor para utilizar nos servidores de sua empresa. A Microsoft, ao ver a possibilidade de sucesso da ideia da concorrente, cria o Javascript, também para seus próprios servidores. A Netscape, ao ver que sua ideia estava sendo copiada, rapidamente desenvolve uma tecnologia do lado do cliente para rodar no próprio navegador. Novamente a Microsoft, ao ver a ideia da concorrente, cria um sistema semelhante. Então a Netscape vai padronizar o Javascript, enviando para o ECMA, para sua incorporação à maioria dos navegadores e a partir daí, surge a EcmaScript (novo nome do Javascript) ao qual é pouco utilizado hoje em dia.

Html

A sigla Html origina-se do inglês, Hyper Text Markup Language (Linguagem de marcação de texto). Não é considerado uma linguagem de programação em si.

O Html, organiza a estrutura de uma página web, tais como: cabeçalho; título; parágrafo; imagem; rodapé e outros. Utiliza-se de etiquetas, ou seja, tags para indicar onde cada elemento será implementado.

Ex: Para inserir um parágrafo, pocedemos da seguinte forma:

“<p>Este é um parágrafo</p>”.

Css

A sigla Css origina-se do inglês Cascading Style Sheet (Folha de Estilo em Cascata). É uma linguagem de estilo e assim como o Html, não é uma linguagem de programação. O Css é responsável por estilizar a página web, tais como: tamanho ou tipo de fonte; espaçamentos; cores de fundo etc. O Css pode ser escrito no mesmo arquivo que o Html ou também em arquivo separado. O seletor será o elemento que queremos estilizar e podem ser tags, classes, identificadores etc. A propriedade é o que de fato será alterado e o valor será a alteração.

Ex:
p {
color: black;
}

Desta forma, todos os parágrafos de página ficarão na cor preta.

JavaScript

As funcionalidades dinâmicas de um site são feitas pelo Javascript, que de fato é uma linguagem de programação. É através do Javascript que se adiciona movimento a uma página web, processa e transforma dados enviados e recebidos e muito mais.

Sempre que se vê um alerta, piscando em um site, é o Javascript o seu responsável e para este exemplo citado, temos o seguinte código:

Alert (‘Helo World’);

Assim como o Css, o Javascript também pode ser escrito dentro de um arquivo Html ou em um arquivo separado.
